# MoneyTracker -- the work still in progress

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.0.7.

## Goals

Let user to track user's expense, income and record all activity where you have spent!

## Features

* Record yours Income, Daily Expenses and Bills, Debt.
* View Graph Analysis by [Chart.js](https://www.chartjs.org/) & [ng2-charts](https://www.npmjs.com/package/ng2-charts) npm package.
* Generate Report with Daily, Weekly or Monthly.

## How it Works

Create a database by using locahost phpmyadmin with mysql. 
Using the httpClient to connect php codes as a server to retreive and save the data
